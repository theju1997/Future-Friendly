Closure

        def outerfunction():
            a=10

                b=20

Closure Example

    def enter_number_outer():
        numbers=[]
    def enter_number_inner(x):
        numbers.append(x)
        print(numbers)
    return enter_number_inner

enter_num=enter_number_outer()
enter_num(3)
enter_num(4)
enter_num(7)
                                  Output
                                  =====
                                  [3]
                                  [3, 4]
                                  [3, 4, 7]
