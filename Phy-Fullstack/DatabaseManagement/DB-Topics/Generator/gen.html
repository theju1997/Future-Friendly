What is Genarator?

Ans: Generator is a function that returns on object(iterator)which we can iterate over (one value at time)
yield keyword: Is going to return the value and pause if we call the same function again the yield resume and returns the value and pause.

    def topten():
        yield 1
        yield 2
    
    values=topten()
    it=iter(values)
    print(next(it))
    print(next(it))
