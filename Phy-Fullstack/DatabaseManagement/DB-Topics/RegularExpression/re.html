Regular Expression(re)

Regular expression is a pattern by using special characters we can find the data or we can verify the data.

Re-Functions

  1. re.match()

    2. Returns a Match object if there is a match first word in the string

  2. re.search()

    3. Returns a Match object if there is a match anywhere in the string

  3. re.findall()

    4. Returns a list containing all matches

  4. re.split()

    5. Returns a list where the string has been split at each match

  5. re.sub()

    6. Replaces one or many matches with a string

Example on Functions

            Example 
            ========
        #Example for match() in regular expression

            import re
            data='lakshmi is python faculty 12pm  batch or your favourite avinash student'
            pattern='lakshmi'
            result=re.match(pattern,data)
            print(result)

        #Example for search() in regular expression

            import re
            data='lakshmi is python faculty 12pm  batch or your favourite avinash student'
            pattern='python'
            result=re.search(pattern,data)
            print(result)

        #Example for findall() in regular expression

            import re
            data='lakshmi is python faculty 12pm  batch or your favourite avinash student python '
            pattern='python'
            result=re.findall(pattern,data)
            print(result)

        #Split the string at every white-space character:
            import re
            txt = "The rain in Spain"
            x = re.split("\s", txt)
            print(x)

        #Replace all white-space characters with the digit "9":

            import re
            txt = "The rain in Spain"
            x = re.sub("\s", "9", txt)
            print(x)
            
                           Output
                       ==============
            re.Match object; span=(0, 7), match='lakshmi'
            re.Match object; span=(11, 17), match='python'
            ['python', 'python']
            ['The', 'rain', 'in', 'Spain']
            The9rain9in9Spain
