Return statement
  1. When ever control is reached to return keword in the funtion
  2. Function returns by default none and end the function.
  3. We can return multiple return statements.
  4. Once first return statement reached and ignores the remaining statements.
  5. A return statement is used to end the execution of the function call and “returns” the result
Note

we call return as many time we want but it will consider only first retun value

3 types of example

                Eg 1:-
                def addition(a,b):
                      c=a+b
                 return c
                x=addition(20,30)
