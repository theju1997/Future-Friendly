Recursion function

Explination with example

    •	It means that a function calls itself.This has the benefit of meaning that you can loop through data to reach a result.
    •	Recursion is the process of defining something in terms of itself.
    •	Calling the function by itself is called recursive invocation.
    •	Recursive invocation leads the infinity.
    •	To restrict the invocation, we are using while loop in below example.

Example

            Eg:-
            i=1
            def func():
                    global i
                    print(‘hi’)
                     while i<5:
                        i=i+1
                    func()
                func()
