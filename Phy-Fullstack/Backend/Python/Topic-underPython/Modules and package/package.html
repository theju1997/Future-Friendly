Package

A package is a way of organizing related modules into a single directory hierarchy.

Example

        my_package/
        |-- __init__.py
        |-- module1.py
        |-- module2.py
        |-- subpackage/
            |-- __init__.py
            |-- module3.py
        
            
my_package/__init__.py 

            This file can be empty, or it can contain package-level initialization code.

            
my_package/module1.py

            def function_in_module1():
            print("Function in module1")

            
my_package/module2.py

            def function_in_module2():
            print("Function in module2")
        

            
my_package/subpackage/module3.py

            def function_in_module3():
            print("Function in module3")

            
Now, we can use this package in another Python script

            # main_script.py

            # Importing the entire package
            import my_package
            
            my_package.module1.function_in_module1()
            my_package.module2.function_in_module2()
            my_package.subpackage.module3.function_in_module3()


            
Explination of package program
               
In this example, my_package is a package containing two modules (module1 and module2) and 
                a subpackage (subpackage) with a module (module3). The main_script.py file demonstrates 
                how to use this package by importing specific modules and calling functions from them.
