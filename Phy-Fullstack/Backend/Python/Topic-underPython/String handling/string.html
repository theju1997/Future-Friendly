String Handling

Explination

String is a group of Character or sequence of Characters .

  • In application we assign by using single or double quotes
  • Triple quotes are used for multiple line or comments
  • String is a immutable (Instd of changing value it will creat new object so)
  • Every string is a index
  • Every String contains positive index and negative index
  • Postive index start with '0'
  • Negative index start with '-1'

Example

name='Theju'
name=name+'bs'
name="Sajina"
