Steping value

If we requires particular part in the given character we can get the element by using steping value

syntax

name [1:9:2]
1 is starting value is included value
9 is ending value is excluded value
2 is steping value

Example

name='tejashwini'

name[0:9:2]
'tjswn'

name[0:10:1]
'tejashwini'

name[0:10:3]
'tawi'

name[0:9:-1]
''

name[9:0:-1]
'iniwhsaje'

name[0:0:-1]
''

name[::-1]
'iniwhsajet'

name[::]
'tejashwini'

name[3:6:1]
'ash'
