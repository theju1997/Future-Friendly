Identity operator

Explination

Is

is : returns true if both address are same

Is not

is not: returns true if both address are not same

Example for is

a=12
id(a)
140727831487624

b=13
id(b)
140727831487656

a is b
False

Both address are different so false

Example for is not

x=20
id(x)
140727831487880

y=20
id(y)
140727831487880

x is y
True

Both address are same so True
