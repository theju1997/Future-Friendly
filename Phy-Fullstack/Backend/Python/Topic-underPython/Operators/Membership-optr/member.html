MemberShip Operators("in") and ("not in")

Explination

in

in :memebership operator if any element is found in the character it will return true(operator symbol is "in")

not in

not in : memebership operator if any element is not found in the character it will return True(operator symbol is "not in")

Example

In operator

'p' in 'python'
True

>>> 'y' in python (if we give without colon we will get error)
 Traceback (most recent call last):
File "", line 1, in

'y' in python
NameError: name 'python' is not defined

>>> 'y' in 'python'
True

>>> 'k' in 'python'
False

not in operator

>>> 'k'not in'python'
True
