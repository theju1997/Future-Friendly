Frozen Set

Frozen set is a immutable object(we can,t Modify , we cant update, we cant delete)

Example

    fruits={"apple","Banana","Cherry","Kiwi","apple"}
frozenset(fruits)
frozenset({'Banana', 'Kiwi', 'Cherry', 'apple'})

basket=frozenset(fruits)
