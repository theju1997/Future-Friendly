For loop on Dictionary

In dictionary we can use only for loop,we cant use while loop becouse indexing concept is not there

Examples for loop

      1)  Cloths={'pant':800,'shirt':600,'saree':1000,'tops':500 ,'shorts':350}
        
      for i in Cloths:
            print('to print keys:',i)
            
Output for keys

            to print keys: pant
            to print keys: shirt
            to print keys: saree
            to print keys: tops
            to print keys: shorts


        for i in Cloths.values():
            print('to print value:',i)
            
Output for values

            to print value: 800
            to print value: 600
            to print value: 1000
            to print value: 500
            to print value: 350
