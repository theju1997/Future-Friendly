Crud/Curd Operation

        C-Create  key value
        r-read--Fetching key value
        u-modify key value
        d-Delete (remove) key value.

Examples on operation

Create

  
        y=dict(btech=2010, mtech=2012, phd=2023)

        y

        {'btech': 2010, 'mtech': 2012, 'phd': 2023}

Modify

  
        my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
        my_dict
        {'name': 'John', 'age': 25, 'city': 'New York'}
        my_dict['age']=26    ("Note:we will get in output least updated value")
        my_dict
        {'name': 'John', 'age': 26, 'city': 'New York'}
