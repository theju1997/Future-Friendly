List

Definition and example

  1. List is a data structure act as a container object
  2. List is a mutable object

    3. We can add the elemet, modify the element, delete the element

  3. List is a order sequence value
  4. List is a Growable and Shrinkable
  5. List have a index with postive index and negative index.
  6. List is a Homogeneous elemets and also Hetrogenous element
  7. List allowes duplicate values
