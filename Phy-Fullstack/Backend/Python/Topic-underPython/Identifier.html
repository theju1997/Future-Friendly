Identifier

Defination

Identifier is a name given to the variable or object

Rule of Identifier

Rule 1: We can write identifier name in lowercase , uppercase ,letters then digits,mixed case,underscore
Rule 2:Digits allowed but not from the begining
Rule 3: space is not allowed
Rule 4:Key word are not allowed
Rule 5:Special symbold are not allowed

Example

List of Valid identifier

  • sony="lower case"
    • print(sony);
    lower case

  • >SONY="Upper case"
    • print(SONY)
    Upper case

  • TheJu="Mixed Case"
    • print(TheJu);
    Mixed Case

  • TheJu123="Mixed case with digits"
    • print(TheJu123);
    Mixed case with digits

  • T_123="underscore with digits"
    • print(T_123);
    underscore with digits

    List of InValid identifier

  • #theju="value" not valid
  • @sony="value" not valid
  • R aju="space not available" not valid
