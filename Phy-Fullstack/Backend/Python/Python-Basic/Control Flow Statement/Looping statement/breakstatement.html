Break Statement

  • Break statement is used in looping statements
  • Break statement is exit from the loop
  • When ever we want break in the looping statement we will use break

    • Example

            i=1
        while i<=10:
            print(i)
            if i==5:
                break;
                i=i+1
            print('end of program')    
                                      Output
                                      --------
                                      1
                                      2
                                      3
                                      4
                                      5
                                      end of program
