While loop

In while loop we should give starting point and ending point

In while loop index is very important

syntax

        while condition:
            statement1
            statement2
                    :
                    :
            main statement

Example

        #Increment
         ----------
        print('generating 1 to 10 using while loop')
        i=1
        while i<=10:
            print(i)#non stop loop it will come 
            i=i+1#to stop we gave this
        print('====================================')
        print('generating 2 increment')
        i=1
        while i<=10:
            print(i) 
            i=i+2
        print('====================================') 

        #Decrement
        -----------
        print('generating Decrement')
        i=10
        while i>=1:
            print(i)
            i=i-1
        print('====================================') 
        
        #Even number
        -------------
        print('creating even number')
        i = 1
        while i <= 100:
            if i % 2 == 0:
                print(i)
            i += 1
        
        print('====================================')
