Nested IF

Explination

Nested if means Inside if block one more if block

Syntax

        if condition:
            block of statement
        if condition:
            block of statement
        else:
            block of statement

Example 


                #Example1

                age=18
                graduate=True
                licence=True
                if age>16:
                    print('You are eligible for driving test')
                if graduate:
                    print('Very good have a great future')
                if licence:
                    print('Attend your driving test')
                print('your not passed the test try it once')
                print('============================================')

                                                                OUTPUT
                                                                -----
                                                                You are eligible for driving test
                                                                Very good have a great future       
                                                                Attend your driving test
                                                                your not passed the test try it once
