If ELSE(if else)

Explination

if condition is true then execute the true block
else execute the false block

Syntax

        if condition:
            block of statement
        else:
            block of statement

Example 

                #Example 1
                
                if 10>5 :
                    print("10 is greater")
                else:
                    print("condition false")
                                         
                                        Output
                                        -------
                                        10 is greater
                ===============================================                        
                #Example 2

                a=35
                b=32
                if a>b:
                    print("a>b : yes")
                elif b>a:
                    print("b>a : No")
                else:
                    print("No")

                                          Output
                                          ------
                                          a>b : yes
                ===============================================                        
                #Example 3
                
                a=30
                b=34
                if a>b:
                    print("condition false")
                elif b>a:
                    print("condition true")
                else:
                    print("No condition")
                     
                                          Output
                                          ------
                                          condition true
