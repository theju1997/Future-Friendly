reduce()

Definition with Example

Reduce is a higher-order function that is used to apply a function of two arguments cumulatively
to the items of an iterable, from left to right, so as to reduce the iterable to a single cumulative result.

syntax

reduce(function, iterable)

OR

Reduce(function, object)

Example

Normal method

        import functools
        a=[3,4,5,6,7]
        def add(x,y):
            return(x+y)
        output=functools.reduce(add,a)
        print(output)
                              Output
                                  25
    ===================================================
    
By using lambda method
     
    import functools
    a=[3,4,5,6,7]
    output=functools.reduce(lambda x,y:x+y ,a)
    print(output)
                            Output
                                25
