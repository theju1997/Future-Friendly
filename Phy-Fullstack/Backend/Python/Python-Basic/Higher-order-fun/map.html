Map()

Definition with Example

Map() is used to iterate the element and modify the element

syntax

map(function,object)

Example

Normal map method 
        z=[1,2,3,4,5]
        def flower(a):
            return a*2
        result=list(map(flower,z))
        print(result)
                                    Output
                                    [2, 4, 6, 8, 10]
    =======================================================
lamda  method
    z=[1,2,3,4,5]
    result=list(map(lambda a:a*2,z))
    print(result)
                                    Output 
                                    [2, 4, 6, 8, 10]
