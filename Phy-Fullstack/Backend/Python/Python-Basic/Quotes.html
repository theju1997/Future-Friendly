Quotes

We will use quotes to give comments

Types of quotes

  1. Single quotes

    2. Application level we can use single quotes.

  2. Double quotes

    3. If we want to write single quotes we can write with in double quotes.

  3. Triple quotes

    4. If we want to write more than one line we can write it in Triple quotes.
