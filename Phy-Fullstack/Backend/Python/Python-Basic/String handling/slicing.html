Slicing

Explination

We can access more than one element in slicing.

Syntax

[start:end:step]
[start:end]
start --->Included
end---->Excluded
Step----> (Default value)

Example

name='Education' name[1:4],
'duc'

name[2:5]
'uca'

name[2:9]
'ucation'

name[1:10]
'ducation'

name[:5]
'Educa'

name[2:]
'ucation'

name[:]
'Education'

name[-5:-1]
'atio'

name[0:-1]
'Educatio'

name[-1:-1]
''

name[-1:5]
''
