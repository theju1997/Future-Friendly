Lamda function(anonymous)

Lambda function also as anonymous function

By using lambda keyword, we can define the anonymous function

We cannot call the lambda functions

Is used execute only one time.

Syntax

lambda arguments : expression

using Normal function example

def addition(a,b):
      return a+b
res=addition(5,6)
print(res)
                        Output:-
                            11
=================================================

using lambda function example

res=lambda a,b: a+b
output=res(10,20)
print(output)

                        Output:-
                            30
=================================================
