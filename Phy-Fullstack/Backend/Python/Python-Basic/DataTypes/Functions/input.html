Input()

Defination

  1. input()is a built in function.
  2. Which accepts input from user and also able to print.
  3. Output of input() always returns string. (if we gave integer value as a input it will return string)

Example

Name=input('Enter your name')
Age=input('Enter your age')
Mobile=input('Enter your phone number')
Address=input('Enter your address')
print(type(Name))
print(type(Age))
print(type(Mobile))
print(type(Address))
print(Name,Age,Mobile,Address)
