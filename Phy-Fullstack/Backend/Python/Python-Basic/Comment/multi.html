Multi-line comment

Multi line comment is indicated by ('''many lines''')

If we want to comment more than one line we should use multi line comment

Example

        '''  commenting below lines its a multi line comment

        print('Hii')
        print('I am learning python')
        #Single line comment symbol is #
        #always write single line comment right side of the code.
        
        a=30 #30 is assigned the value of a
        b=60 #60 is assigned the value of b
        c=a+b 
        print(c)  
        
        '''
