Format method

Definition and example

Format method will help to add some text inside the paragraph or anything

Exmaple

3 ways of format method

------------------------------------------------------------------------------
type1='my name is {}, my age{}'.format('Thejaswini' , 26)
type1
'my name is Thejaswini, my age26'

------------------------------------------------------------------------------
type2='my name is {0}, my age{1}'.format('Thejaswini' , 26)
type2
'my name is Thejaswini, my age26'

------------------------------------------------------------------------------
type3='my name is {x}, my age{y}'.format(x='Thejaswini' ,y=26)
type3
'my name is Thejaswini, my age26'

------------------------------------------------------------------------------
