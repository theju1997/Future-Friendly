Object

What is object?

Physical existance of a class is nothing but object.

Syntax

reference variable=class name(Object)


What is reference variable?

The variable which stores the address of the object is called reference variable.

By using reference variable we can access properties and methods.


What is self variable?

Seft is a default variable which always pointing to current object


Example

For class, variable, object. reference name , self 

        class student:
        def __init__(self,name,rollno,marks):
            self.name=name
            self.rollno=rollno 
            self.marks=marks
        def result(self):
            print('my name is ',self.name)
            print('my rollno is ',self.rollno)
            print('my marks is ',self.marks)
            print('==========================')
       s1=student('theju',57,85)
       s1.result()
       s2=student('Adi',58,85)
