Open

The "open" operation is a fundamental step before you can perform read, write, or other file-related operations.

        
Example for with in the same folder -How to open the file?

        f=open('sample1.txt','r') #for open the file
        print(f.read())
        f.close()
                               Output will print what we written in sample1 File

        
Example for other folder file -How to open the file?

        f=open('c:/user/pc/desktop/sample2.txt','r') #for open the file
        print(f.read())
        f.close()

                               Output will print what we written in sample2 File
