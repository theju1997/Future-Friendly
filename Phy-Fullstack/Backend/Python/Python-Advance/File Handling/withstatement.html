With Statement

The with statement can be used while opening a file , we can use the operation statements

With in a block

The advantage of with statement is it will take care closing of the file

syntax

    with open(path,mode)as  alias:
        statement1
        statement2
        ;
        ;
        ;
        statement n

Example

with open('theju.txt','w')as f:
    f.write('Advantage of with statement')
    f.write('It will close the file automatically')

                                                        Output
                                                        -------
                                                        Advantage of with statement 
                                                        It will close the file automatically
