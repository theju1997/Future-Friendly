Methods of read

  1. read()

    2. read all the lines as string

  2. read(n)

    3. reads the number of characters

  3. readline()

    4. reads the one line at a time

  4. readlines()

    5. It reads all the lines and stores in to list

#Example for readline()

        x=open('sample.txt','r')
        line1=x.readline().strip()
        line2=x.readline()
        print(line1)
        print(line2)
        x.close()

                  output
                 --------
                 Hii I'm  Theju
                 I am learning python

            
        
#Example for readlines()
        x=open('sample.txt','r')
        line1=x.readlines()
        print(line1)
        x.close()
                    
                        output
                        --------
                        ["Hii I'm  Theju\n", 'I am learning python\n', 'This is file handling topic']
