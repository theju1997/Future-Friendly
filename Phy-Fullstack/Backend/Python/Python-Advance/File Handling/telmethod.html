Tell and Seek Method

Tell method

Tel the position of the file pointer

Seek method

It is used to move the curser in required position

Example for Tell and Seek

        f=open('student.txt','w')                         #To open and write the file(to create file)
        f.write('All students are Stupids')               #To write the file
        f.close()                                         #To close the file
        with open('student.txt','r+')as f:                #To read and write the file
            text=f.read()                                 #To read the file
            print(text)
            print('the current cursor position',f.tell())  #to ask the cursor place
            f.seek(17)                                     #to place the cursor in 17 th place
            print('the current cursor position',f.tell())
            f.write('Gems!!')                              #to change the word from 17th place
            print('the current cursor position',f.tell())
            f.seek(0)                                      #to move the cursor begining place
            text=f.read()
            print('data after modification')
            print(text)                                    #to print the text
                  
                       Output
                       -------
                       All students are Stupids
                       the current cursor position 24
                       the current cursor position 17
                       the current cursor position 23
                       data after modification
                       All students are Gems!!s
