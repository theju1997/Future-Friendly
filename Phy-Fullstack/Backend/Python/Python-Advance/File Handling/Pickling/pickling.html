Pickling

Serialization

The concept of storing the python object into the binary file is called pickling or serialization

Example

        import pickle

        data=[['Shivanshis',7398,200],['Shanthi',7368,300],['Theju',7598,600]]
        print(data)
        f=open('picex.txt','wb')
        pickle.dump(data,f)
        f.close()

De-Serialization

Reading data from binary file and converting to the python object is called unpickling or deserialization.

Example

    import pickle

    data=[['Shivanshis',7398,200],['Shanthi',7368,300],['Theju',7598,600]]
    f=open('picex.txt','rb')
    data=pickle.load(f)
    print(data)
    f.close()
