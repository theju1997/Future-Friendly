Write

Two methods in write

  1. write()
  2. write lines()

--write()

write is used write text into the file .

Example

    f=open('theju.txt','w')
    f.write('I am developer \n')
    f.write('I am a python developer')
    f.close()
    Note: it will create the file.
                                Output
                                -------
                                I am developer
                                I am a python developer (it will print in the file theju.txt)

--write lines()

write lines used to write lines into the file by taking list of lines
(means it will print line by line if we gave in list)

Example

        f=open('theju.txt','w')
        list1=['I am a theju \n','I like to do coding \n',]
        f.writelines(list1)
        f.close()
                                       Output
                                       =======
                                       I am a theju
                                       I like to do coding
