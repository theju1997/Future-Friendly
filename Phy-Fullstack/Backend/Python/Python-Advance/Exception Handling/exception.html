Runtime error are synticatcly free . it occurs during execution of the program.

For different input.

Runtime errors are called exception.

Exception Handling have 3 keys

  1. Try
    2. Keep the code in try block which is going to raise an exception. 
            syntax:
        try:
            block
  2. Except
    3. except block catch the exception and handles the exception. 
            syntax:
        except(exception):
            block
  3. Finally

    4. Always execute if we get erro or not get error

    If exception occured or not occured finally block is always executes

            syntax 
        finally:
            block
