Abstraction

It is used to hide the complexity of implementation and highlighting the services is know as abstraction

OR

It is defined as a process of handling complexity by hiding unnecessary information from the user.

Abstract Method

Any method with out implementaion with decorator abstract method is called abstract method

Abstract method is available in abc module

Abstract Class

Any class which inherits from AbC

ABC class is available in abc module

When we can creat object

If class contains abstract method we can create object

If class contains abstract class we can create object

If class contains abstract class and also abstract method we can't creat object

Example-1

        from abc import ABC,abstractmethod
        class wifi(ABC):
            @abstractmethod
            def network(self):
                pass
        
        class jio(wifi):
            def network(self):
                print('5g wifi with x')
            def network1(self):
                print('5g wifi with xy')
        
        class Airtel(wifi):
            def network(self):
                print('6g wifi with xyz')
        
        air=Airtel()
        air.network()
        
        j=jio()
        j.network()
        j.network1()

                                   Output
                                   -------
                                   6g wifi with xyz
                                   5g wifi with x
                                   5g wifi with xy

Example-2

        #By using two methods 
        from abc import ABC,abstractmethod
        class wifi(ABC):
            @abstractmethod #method
            def network(self):
                pass
            @abstractmethod #method
            def charges(self):
                pass
            def enjoy(self):#normal method
                print('enjoy is norma method we can implement without creating class')
        
        class jio(wifi):
            def network(self):
                print('5g wifi with x')
               
        
        class Airtel(jio):
            def charges(self):
                print(' 2 paise tariff')
        
        air=Airtel()
        air.network()
        air.charges()
        air.enjoy()

                                            Output
                                            -------
                                            5g wifi with x
                                            2 paise tariff
                                            enjoy is norma method we can implement without creating class
