Static Variable

Static variables are often used to store class-level information that is shared among all instances.
They are defined within the class but outside of any instance method.
You can access static variables using the class name or the instance name.

Static Method

  • Methods which performs operation on external data
  • It can also perform operation on class data
  • No need to pass object or class reference
  • Created by using decorator '@staticmethod'
    • (Face book login and password page is common for everyone)
