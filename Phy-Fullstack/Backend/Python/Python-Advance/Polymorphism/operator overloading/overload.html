Operator overloading

Work flow of operator overloading

dir(int)(to check magic method of operator)
step1:Check datatype left operand
step2:go in that class
step3:call __add__()method 


    
Example

    num1='hello'
    num2='theju'
    print(num1+num2)
    print(num1.__add__(num2))
    print(str.__add__(num1,num2))

                                            Output
                                            -------
                                            hello theju
