Constructor Overloading

Constructor overloading in Python refers to defining multiple constructors within a class, each with a different set of parameters, allowing the class to be instantiated in various ways.

Example

            class Calculator:
            def __init__(self, a, b=0, c=0):
                print('Sum of parameters:', a + b + c)
        
        c = Calculator(10, 20, 30)

                                                        Output
                                                        ------
                                                        Sum of parameters: 60
