Method overloading

If two or more methods have same method with different parameter is know as method overloading

Example

            class calculator:
            def add(self,a,b):
                print('i am a two parameters of addition',a+b)
            def add(self,a,b,c):
                print('i am a three parameters of addition',a+b+c)
            def add(self,a,b,c,d):
                print('i am a four parameters of addition',a+b+c+d)
        c=calculator()
        c.add(10,20,30,40)
                                        Output
                                        -------
                                        i am a four parameters of addition 100
