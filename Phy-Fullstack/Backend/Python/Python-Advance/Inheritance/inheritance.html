Inheritance

Acquiring the properties from the parent class(existing class) to child class(new class)

Example 1

    class Ravikanth:
        house_name = 'Theju nilaya'
    
        def tmane(self):
            print(self.house_name)
    
    class Suryakanth:
        house_name1 = 'Adi nilaya'
    
        def amane(self):
            print(self.house_name1)
    
    r1 = Ravikanth()
    print(r1.house_name)
    r1.tmane()
    
    s1 = Suryakanth()
    print(s1.house_name1)
    s1.amane()
                                                    Output
                                                    ------
                                                    Theju nilaya
                                                    Theju nilaya
                                                    Adi nilaya
                                                    Adi nilaya       
                                                    

Example 2

class Employee:

    def display(self):

    bonus=5000

m=Manager()
