Single Inheritance

The concept of acquiring the properties from one class into another class is called single inheritance

Example

    class theju:
        def m1(self)
        print('Single inheritance')
    t=theju()
    t.m1()
    t.__str__()    
    #str from help of object we have many magic method we can use any one#

                                                                        Output
                                                                        -------
                                                                        single inheritance
                                                                        <__main__.Theju object at 0x00000237368157F0>
