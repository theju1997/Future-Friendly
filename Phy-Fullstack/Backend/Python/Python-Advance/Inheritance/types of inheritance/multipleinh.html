Multiple Inheritance

Inheriting properties from multiple classes into single class

Multiple parent to single child 
    class ramu:
        def r1(self):
            print('race bike')
    class krishna:
        def k(self):
            print('Mini-cooper')
    class theju(ramu,krishna):
        def th(self):
            print('swift')
    t=theju()
    t.th()
    t.k()
    t.r1()

                                    Output
                                    -------
                                    swift
                                    Mini-cooper
                                    race bike
