Multilevel Inheritance

The concept of inheriting the properties from multiple class one after another is called multilevel inheritance

Example

        
        class grandpa:
            def G1(self):
                print('object class is a parent of grandfather(by default if dnt have any parent means, object class itself parent)')
        class Father(grandpa):
            def F1(self):
                print('Grand father is a parent of father')
        s=Father()
        s.F1()
        s.G1()
        print(s.__str__())
        
                                                             OUTPUT
                                                             -------
                                                       Grand father is a parent of father
                                object class is a parent of grandfather(by default if dnt have any parent means, object class itself parent)
                                                            <__main__.Father object at 0x00000251E65A5AF0>
