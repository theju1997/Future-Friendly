Constructor Inheritance

In object oriented programming , constructor inheritance refere to the concept of a (or derived class)inheriting the constructor of its super class(or base clas)

Example

        class Father:
            def __init__(self):
                print('father constructor called')
                self.vehicle='Bike'
                print(self.vehicle)
        class son(Father):
            pass
        s=son()

                                                    Output
                                                    -------
                                                    father constructor called
                                                     Bike
