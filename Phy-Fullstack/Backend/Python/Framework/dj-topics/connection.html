How to connect DJango to sqlite ,MySql

TO CONNECT DJANGO TO SQLITE3

Step 1: Open models.py file
	Create one class 
	class industry(models.Model):
    actor_name = models.CharField(max_length=30)
    first_movie = models.CharField(max_length=30)
    second_movie = models.CharField(max_length=30)
    release_date = models.DateField()

Step 2:Run to commands
 	Python manage.py makemigrations
	Python manage.py migrate

Step 3:Setting.py 
        No need to change connection by default it connected to sqllite3

Step 4:After the makemigrations we can chek the file migration it will give created model list

Step 5:After the migrate we can check in database it will show the created table.


TO CONNECT DJANGO TO MYSQL


Step 1:Make sure models are created or not 

Step 2:   In models.py
	In this file we should create one class and we should add the contents
	from django.db import models

# Create your models here.
class industry(models.Model):
    actor_name = models.CharField(max_length=30)
    first_movie = models.CharField(max_length=30)
    second_movie = models.CharField(max_length=30)
    release_date = models.DateField()

Step 3:Settings.py
Change connection sqllite3 to MySql
Go to mysql command login with password
Create one database name
	DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',

}

Step 4: Run the server using two commands
	1)python manage.py makemigration(if u get error solve the erro by using google)

	2)python manage.py migrate

Step 5: Go to mysql check the table created are not  
	Show tables;(to check the table is created or not)
	select *from drama_industry; (we will get empty set)
	describe drama_industry;(we will get complete created table)

My Sql synatax

  1. Show databases;(to see all database names)
  2. Use database;(to use required database)
  3. show tables;(to see created table name)
  4. select * from tablename;(we can see added objects inside the table)
  5. describe info_peoples;(we can check fields)
