Introduction of form

Form Type

  • An HTML form is used to collect user input. The user input is most often sent to a server for processing.

  • The HTML form tag element is used to create an HTML form for user input

  • The form tag element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

  • The HTML (input tag)for box element is the most used form element. An input tag element can be displayed in many ways, depending on the type attribute.
