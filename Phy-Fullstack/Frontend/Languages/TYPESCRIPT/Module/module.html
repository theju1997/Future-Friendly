MODULE

We have 2 types under Module

1) Static Module

A static method in JavaScript is a method that has a static keyword prepended to itself.

The static keyword defines a static method or field for a class, or a static initialization block

>Example for static module Project1(Static Calculator)

2) Dynamic Module

Dynamic module imports are very similar to static module imports, but the code is loaded only
when it is needed instead of being loaded right away.

>Example for dynamic module Project2(Dynamic Calculator)

We have 2 Default types under Dynamic module

1) Synchronise
