ARRAYS

Explination about arrays

An array is a special variable, which can hold more than one value

Why Use Arrays?

If you have a list of items (a list of car names, for example),
storing the cars in single variables could look like this:
let car1 = "Swift";,
let car2 = "Volvo";
let car3 = "BMW";


Note: If similar type of data is there more than countless we will use array.
arrays will start from 0,1,2(key values)

Example....

In arrays for more than one item we can use this code :
const array_name = [item1, item2, ...];
