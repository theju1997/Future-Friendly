Libraries

Explination:

Frameworks and libraries are software packages that are prewritten/reusable code
modules which can be imported into a project. These standardized front-end
technologies help front-end developers accelerate the development process
without spending copious time coding every object or function from scratch.

What are libraries in frontend

They are typically used to create the user interface of a web application, including the layout, design, and interactivity.
Frontend libraries are often written in JavaScript, and they
provide a range of features such as UI components, animations, data visualization, and more.
